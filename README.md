# Patient Engagement Services

Smart contract system that enables comprehensive patient communication and engagement tracking for healthcare providers. Handles appointment reminders, satisfaction surveys, health education delivery, and outcome monitoring.

## What's Included

* Patient registration with communication preferences
* Appointment scheduling and automated reminder tracking  
* Post-visit satisfaction survey creation and response collection
* Health education content delivery with engagement metrics
* Dynamic patient engagement scoring based on interactions

## Technical Notes

Implemented using Clarity's map data structures for efficient patient, appointment, and survey management. The contract maintains separate tracking for different engagement activities, allowing providers to build comprehensive patient interaction histories. Each patient gets assigned to their provider, ensuring proper data isolation and access control.
